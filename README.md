# 토토 슬립 생성기

GitHub Pages에 바로 올릴 수 있는 정적 HTML 버전입니다.

## 파일 구성

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## 배포 방법

1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소 루트에 업로드합니다.
3. 저장소의 **Settings → Pages** 로 이동합니다.
4. Branch를 `main`, 폴더를 `/ (root)` 로 선택합니다.
5. 저장 후 생성된 Pages 주소로 접속합니다.

## 지원 입력 형식 예시

```text
1조합
099무 104승 111패 122무 148승 (10,000원) (99.464배)

2조합
107무 119패 130패 159승 171무 (10,000원) (99.921배)
```

```text
1조합 099무 104승 111패 122무 148승 1만원 99.464배
2조합 107무 119패 130패 159승 171무 (1만원) (99.921배)
```

## 참고

- 슬립 복사는 브라우저가 이미지 클립보드 복사를 지원할 때 동작합니다.
- 일부 환경에서는 `PNG 저장` 버튼이 더 안정적입니다.
